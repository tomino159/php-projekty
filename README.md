# 🐘 Moje PHP Projekty

Kolekcia PHP projektov vytvorených počas učenia sa PHP vývoja.

## 📁 Projekty

| Súbor | Popis |
|---|---|
| `bmi.php` | Kalkulačka BMI s hodnotením |
| `kalkulacka.php` | Funkčná kalkulačka (sčítanie, odčítanie, násobenie, delenie) |
| `triedna_kniha.php` | Správa žiakov a známok s funkciami |
| `nasobilka.php` | Násobilka pomocou for cyklu |
| `registracia.php` | Registračný formulár s validáciou |
| `login.php` | Prihlasovací systém so sessions |
| `dashboard.php` | Chránená stránka po prihlásení |
| `spravca.php` | CRUD správa používateľov s MySQL |
| `kviz.php` | Interaktívny kvíz s vyhodnotením |
| `teplota.php` | Prevodník teplôt (°C → °F, K, R) |
| `kontakt.php` | Validovaný kontaktný formulár |
| `nakup.php` | Nákupný zoznam s výpočtom celkovej ceny |
| `horoskop.php` | Generátor náhodných horoskopov podľa znamenia |
| `farba.php` | Generátor farby dňa z aktuálneho dátumu |
| `pingpong_score.php` | Skóre tabuľka pre pingpong so SESSION |
| `caesar.php` | Šifrovanie textu pomocou Caesar šifry |
| `losovac.php` | Náhodné rozdelenie hráčov do dvoch tímov |
| `stopky.html` | Funkčné stopky s milisekundami |

## 🚀 Spustenie

Vyžaduje [XAMPP](https://www.apachefriends.org/) s Apache a MySQL.

1. Skopíruj súbory do `C:\xampp\htdocs\moje_php\`
2. Spusti Apache a MySQL v XAMPP
3. Otvor `http://localhost/moje_php/`
